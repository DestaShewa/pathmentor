## 🎯 How to Use

### Send a File
1. Click the **📎 paperclip icon**
2. Select one or more files (up to 5)
3. Preview appears showing filename and size
4. Optionally type a text message
5. Click **Send** button
6. File uploads and appears in chat

### Record Voice Message
1. Click the **🎤 microphone icon**
2. Allow microphone access (if prompted)
3. Speak your message
4. Watch the timer count up (0:01, 0:02, ...)
5. Click **Stop** button
6. Voice file appears in attachments
7. Click **Send** button
8. Voice message appears with play button

### Send Multiple Items
1. Attach multiple files (images, documents, etc.)
2. Record a voice message
3. Type a text message
4. All items send together in one message!

## 🐛 Troubleshooting

### "Failed to send message"
- Check backend server is running (port 5001)
- Check file size is under 25MB
- Check file type is supported

### "Could not access microphone"
- Click "Allow" when browser asks for permission
- Check microphone is not used by another app
- Try refreshing the page

### Files not uploading
- Check file type is supported
- Check file size is under 25MB
- Check internet connection

### Voice not recording
- Allow microphone permission
- Check microphone is working
- Try a different browser
