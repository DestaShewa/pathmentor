🚀 HOW TO RUN

Backend:
  cd backend
  node server.js
  → Runs on http://localhost:5001
  → Connects to MongoDB Atlas (falls back to local if Atlas unavailable)

Frontend:
  cd frontend
  npm run dev
  → Runs on http://localhost:8080

Seed Test Data:
  cd backend
  node scripts/seedData.js
